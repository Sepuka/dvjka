В секции db

DBname = freelance (имя БД)
DBuser = root (пользователь)
DBpass = '1' (пароль)
PREFIX = DVJK_ (оставляем как есть)

[site]
host = dvjk (имя хоста без http, например srv28670.ht-test.ru)
# Время которое сайт помнит пользователя (в сек.) - 100 дней

В секции cron все уже настроено. Запуск задач:
php cron.php fraudster # 48 часовое блокирование мошенников
php cron.php noObtained # Автоматическое подтверждение раз в сутки платежей (если получатель заснул)
php cron.php noPaymentBlock # создал заявку и не оплатил? Платеж удаляется, клиент блокируется
